# ThinkTwice Assets

Image assets for the ThinkTwice riddles app.

## Structure

- `riddles/` - riddle illustrations (light and dark versions)

## Usage

Images are loaded via GitHub Raw URLs: https://raw.githubusercontent.com/MaksimNigmatullin/thinktwice-assets/main/riddles/strange_ride_001_light.webp


## Auto-sync

Use `push.ps1` to sync images from `composeResources/drawable`.
